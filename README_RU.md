# Сводка по каналу Telegram

> Быстро собирайте ключевые метрики вашего канала Telegram с помощью API для разработчиков и библиотеки Telethon.

## Обзор проекта

Скрипт на Python автоматически получает из канала Telegram статистику по сообщениям, просмотрам, пересылкам и медиа, формируя удобную для чтения сводку.

## Быстрый старт

### 1. Установка зависимостей

Убедитесь, что Python установлен, затем выполните:

```bash
pip install telethon
```

### 2. Настройка учётных данных разработчика

1. Откройте [страницу разработчика Telegram](https://my.telegram.org/apps).
2. Получите свои `api_id` и `api_hash`.
3. В файле `tg_ru.py` замените значения `api_id` и `api_hash` на полученные.

### 3. Параметры канала

- В переменную `channel` запишите имя пользователя или ссылку на канал, статистику которого хотите собрать.
- При большом объёме сообщений увеличьте `BATCH_SIZE`, чтобы избежать ограничений по частоте запросов.

### 4. Запуск скрипта

Запустите скрипт командой:

```bash
python tg_ru.py
```

При первом запуске введите номер телефона с кодом страны (например, `+13333333333`). Затем в Telegram придёт код подтверждения — введите его в консоль. После этого скрипт подготовит сводку по каналу.
